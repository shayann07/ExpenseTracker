# Expense Tracker

## Overview
Expense Tracker is a simple and intuitive Android app that helps you track your daily expenses and incomes. With Expense Tracker you can log transactions, categorize them, set monthly budgets, and visualize your spending patterns. The goal is to provide a clean, user‑friendly interface so you can maintain financial awareness on the go.

## Features
- **Add & Manage Transactions:** Add income or expense entries with amount, category, date, and description. Edit or delete entries as needed.
- **Category Management:** Choose from predefined categories or create your own custom categories to organize your finances.
- **Budget Goals:** Set monthly budget goals for different categories and monitor your progress throughout the month.
- **Visual Insights:** View your spending and income trends through charts and summaries to help identify habits and plan accordingly.
- **Search & Filter:** Easily search transactions by keyword or filter by category, date range, or type (income/expense).
- **Local Data Storage:** Your data is stored locally using Room (SQLite) ensuring privacy and offline access.
- **Modern UI & Architecture:** Built using Kotlin with MVVM architecture, Jetpack components such as LiveData, ViewModel, and Material Design to provide a responsive and consistent user experience.

## Tech Stack
| Layer | Tech |
| --- | --- |
| Language | Kotlin |
| Architecture | Model‑View‑ViewModel (MVVM) |
| Database | Room (SQLite) |
| UI | Jetpack Compose / Material Design components |
| Dependency Injection | Dagger‑Hilt |
| Asynchronous | Kotlin Coroutines & Flow |

## Getting Started
1. **Clone the repository**  
   ```bash
   git clone https://github.com/shayann07/ExpenseTracker.git
   ```
2. **Open in Android Studio**  
   • Launch Android Studio and select *Open an existing Android Studio project*.
   • Navigate to the cloned `ExpenseTracker` directory and open it.

3. **Build & Run**  
   • Ensure you have an Android device or emulator configured.
   • Click *Run* ▶️ to build and install the app on your device.

4. **Enjoy!**  
   Start adding your expenses and incomes to keep track of your finances.
